# SE-Channel-schedule

## Guide

First clone this project directory.

```bash
git clone https://github.com/kasun7153/B-Airways-backend.git
```

Install Node Modules

```bash
npm install
```

 Then run the express server.

```bash
npm start
```

### Api Endpoints

* SignUp - POST

```bash
http://localhost:5000/user/register
```
